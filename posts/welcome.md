# Welcome

This is a sample article. Replace it with your own writing.

To add a new post:

1. Create a new file in `posts/` named `your-slug.md` (use lowercase and hyphens).
2. Add an entry to `posts/manifest.json` with `title`, `slug`, and `date`.
3. Deploy as you already do on Vercel; no build step is required.

You can draft in any Markdown editor (Obsidian, VS Code, iA Writer) and paste the file into the repo when you are ready.
