# Curriculum vitae (summary)

This page mirrors the highlights from my CV ([download the PDF](Nestor_Rendon_AI_Data_Science.pdf)). I am **Nestor Rendon, PhD** — AI researcher with a doctorate in data science and statistical modeling, based in **Lugano, Switzerland**.

## Professional profile

I design and implement intelligent systems with an emphasis on modular machine learning pipelines, conversational agents, LLM fine-tuning, and scalable AI in production. My background includes unsupervised learning (especially audio and soundscapes), deep learning for vision, and high-dimensional statistical modeling. I am particularly interested in software architectures for multi-agent systems and in integrating large language models into modular, maintainable systems.

## Selected experience

- **xFarm Technologies (Switzerland), 2020–present** — Vision engineer and data scientist: conversational agents and deep learning for agricultural monitoring; end-to-end ML pipelines (ingestion through deployment); modular integration of vision models in production; monitoring of inference in cloud environments (AWS, GCP); experimentation with multi-agent orchestration; containerization, version control, and structured engineering workflows.

- **University of Antioquia (Colombia), 2021–2024** — PhD researcher in machine learning and statistical modeling: unsupervised methods for soundscape analysis; internal clustering validation using Fréchet distance; modular experimental frameworks for geospatial, audio, and time-series data; statistical modeling of high-dimensional data; peer-reviewed publications.

- **Laboratoire MIA (France), 2022–2023** — Machine learning engineer: benchmarking unsupervised algorithms; modular ML pipelines for ecoacoustic data; interdisciplinary AI research.

- **SUPSI–DTI, Reteca internship (Switzerland), 2020–2021** — Machine learning engineer: deep learning for object detection; data acquisition and preprocessing modules; collaboration with industrial partners.

- **IES and Colombia de Colores (Colombia), 2017–2020** — Software / ML engineer and teacher: applied software projects; ML for rain detection in acoustic recordings; programming instruction.
